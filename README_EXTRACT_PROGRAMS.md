# AegisFlow - Extract Programs

**Transform, Validate, Deliver from a Single TOC**

## Feature Description

Extract PROGRAM list from Excel file generated by MOSAIC_CONVERT, perform statistical analysis, and generate SAS execution script in text file format.

## Main Features

1. **Excel File Analysis**: Read Excel file generated by MOSAIC_CONVERT (Index worksheet)
2. **Program Statistics**: Count each unique PROGRAM value and corresponding tocnumber quantity
3. **Order Preservation**: Arrange by order of first appearance in Excel (maintain original sequence)
4. **Script Generation**: Generate standard %runpgm format SAS script with comments unified at top
5. **Graphical Interface**: Provide file selection dialog, no need to manually modify paths
